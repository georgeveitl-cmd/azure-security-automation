# Azure Security Automation

This repository contains security automation examples focused on Azure identity governance, RBAC auditing, and cloud security monitoring.

The scripts and documentation demonstrate techniques commonly used by security operations teams to review access permissions, identify privileged roles, and monitor identity activity within Azure environments.

## Purpose

The goal of this project is to demonstrate security automation concepts including:

- Azure RBAC auditing
- Identity governance monitoring
- Privileged role identification
- Security reporting automation

All examples are generic lab demonstrations and do not contain any proprietary or production environment information.
